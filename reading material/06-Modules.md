# Modules in NestJS
1. A container where we eep related controllers services and providers for our application.
2. It is Core part of nestjs architecture.
3. Every Nestjs app has at least one module, the root module.
4. A module is a class annotated with @Module() decorator.
5. The @Module() decorator provides metadata that Nest makes use of to organize the application structure.

# Generate a module using nest cli command
```bash
nest g module <module-name>
```
