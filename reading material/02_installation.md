# NestJS Installation

1. Install nested CLI globally using npm:
```bash
npm install -g @nestjs/cli
```

2. Create a new NestJS project:
```bash
nest new project-name
```

3. Change into the project directory:
```bash
cd project-name
```

4. Install project dependencies:
```bash
npm install
```

5. Start the development server:
```bash
npm run start
```

6. Open your browser and navigate to `http://localhost:3000` to see the default NestJS application running.
